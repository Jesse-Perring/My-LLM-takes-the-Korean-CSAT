\[f'(a) = \lim_{h \rightarrow 0} \frac{f(a+h) - f(a)}{h}\]

Given the function \(f(x) = x^3 - 8x + 7\), we need to find the derivative \(f'(x)\) and then evaluate it at \(x = 2\) to find the value of the given limit.
