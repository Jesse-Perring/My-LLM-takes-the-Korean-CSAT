문제 1:
To solve the problem of finding the value of \(\sqrt[3]{5} \times 25^{\frac{1}{3}}\), we start by simplifying each part of the expression.

First, let's consider the term \(25^{\frac{1}{3}}\). We can express 25 as a power of its prime factor: \(25 = 5^2\). Using this, we rewrite \(25^{\frac{1}{3}}\) as \((5^2)^{\frac{1}{3}}\). By applying the exponent rule \((a^m)^n = a^{m \cdot n}\), we get:

\[
(5^2)^{\frac{1}{3}} = 5^{2 \cdot \frac{1}{3}} = 5^{\frac{2}{3}}
\]

Now, let's look at the original expression \(\sqrt[3]{5} \times 25^{\frac{1}{3}}\). The term \(\sqrt[3]{5}\) can be rewritten as \(5^{\frac{1}{3}}\).

So, the expression becomes:

\[
5^{\frac{1}{3}} \times 5^{\frac{2}{3}}
\]

We can combine these two terms because they have the same base. According to the property of exponents \(a^m \times a^n = a^{m+n}\), we add the exponents:

\[
5^{\frac{1}{3}} \times 5^{\frac{2}{3}} = 5^{\frac{1}{3} + \frac{2}{3}} = 5^{\frac{3}{3}} = 5^1
\]

Thus, the value of the expression is simply 5.

Answer: 5

\[
\lim_{h \rightarrow 0} \frac{f(2+h) - f(2)}{h}
\]

First, let's calculate \( f(2) \):

\[
f(2) = 2^3 - 8 \times 2 + 7 = 8 - 16 + 7 = -1
\]

Next, we need to find \( f(2+h) \) by substituting \( x = 2+h \) into the function:

\[
f(2+h) = (2+h)^3 - 8(2+h) + 7
\]

We expand \( (2+h)^3 \):

\[
(2+h)^3 = 2^3 + 3 \cdot 2^2 \cdot h + 3 \cdot 2 \cdot h^2 + h^3 = 8 + 12h + 6h^2 + h^3
\]

Substitute this back into the expression for \( f(2+h) \):

\[
f(2+h) = (8 + 12h + 6h^2 + h^3) - 8(2+h) + 7
\]

Simplify the expression:

\[
f(2+h) = 8 + 12h + 6h^2 + h^3 - 16 - 8h + 7
\]

Combine like terms:

\[
f(2+h) = (8 - 16 + 7) + (12h - 8h) + 6h^2 + h^3 = -1 + 4h + 6h^2 + h^3
\]

Now, calculate \( f(2+h) - f(2) \):

\[
f(2+h) - f(2) = (-1 + 4h + 6h^2 + h^3) - (-1) = 4h + 6h^2 + h^3
\]

We then evaluate the limit:

\[
\lim_{h \rightarrow 0} \frac{f(2+h) - f(2)}{h} = \lim_{h \rightarrow 0} \frac{4h + 6h^2 + h^3}{h}
\]

Simplify the expression by dividing each term by \( h \):

\[
= \lim_{h \rightarrow 0} (4 + 6h + h^2)
\]

As \( h \) approaches 0, the terms \( 6h \) and \( h^2 \) approach 0, so the limit is:

\[
= 4
\]

Thus, the value of the limit is:

\[
\text{Answer: } 4

\[
\frac{a_4}{a_2} + \frac{a_2}{a_1} = 30
\]

\[
\frac{k^4}{k^2} + \frac{k^2}{k} = 30

\[
k^2 + k = 30
\]

\[
k^2 + k - 30 = 0
\]

The function \( f(x) \) is defined as:
\[ 
f(x) = 
\begin{cases} 
5x + a & \text{if } x < -2 \\ 
x^2 - a & \text{if } x \geq -2 
\end{cases} 
\]

For \( f(x) \) to be continuous at \( x = -2 \), the left-hand limit as \( x \) approaches \(-2\) from the left must equal the right-hand limit as \( x \) approaches \(-2\) from the right, and both must equal \( f(-2) \).

1. **Left-hand limit**: As \( x \) approaches \(-2\) from the left (\( x \to -2^- \)), we use the expression \( 5x + a \). Substituting \( x = -2 \) into this expression, we get:
   \[
   \lim_{x \to -2^-} f(x) = 5(-2) + a = -10 + a
   \]

2. **Right-hand limit**: As \( x \) approaches \(-2\) from the right (\( x \to -2^+ \)), we use the expression \( x^2 - a \). Substituting \( x = -2 \) into this expression, we get:
   \[
   \lim_{x \to -2^+} f(x) = (-2)^2 - a = 4 - a
   \]

For continuity at \( x = -2 \), these two limits must be equal:
\[
-10 + a = 4 - a
\]

Solving this equation for \( a \), we add \( a \) to both sides:
\[
-10 + a + a = 4 - a + a
\]
\[
-10 + 2a = 4
\]

Next, add 10 to both sides:
\[
2a = 14
\]

Finally, divide both sides by 2:
\[
a = 7
\]

Thus, the value of the constant \( a \) that makes the function \( f(x) \) continuous for all real numbers is \( 7 \).

Answer: 7

To find the value of \( f'(1) \) for the function \( f(x) = (x^2 + 1)(3x^2 - x) \), we need to differentiate \( f(x) \) using the product rule. The product rule states that if \( u(x) \) and \( v(x) \) are functions of \( x \), then the derivative of their product is given by:

\[
f'(x) = u'(x)v(x) + u(x)v'(x)
\]

In this case, we identify \( u(x) = x^2 + 1 \) and \( v(x) = 3x^2 - x \).

First, we differentiate \( u(x) = x^2 + 1 \):

\[
u'(x) = \frac{d}{dx}(x^2 + 1) = 2x
\]

Next, we differentiate \( v(x) = 3x^2 - x \):

\[
v'(x) = \frac{d}{dx}(3x^2 - x) = 6x - 1
\]

Now, we apply the product rule:

\[
f'(x) = u'(x)v(x) + u(x)v'(x)
\]

Substituting the expressions we found:

\[
f'(x) = (2x)(3x^2 - x) + (x^2 + 1)(6x - 1)
\]

Simplify each term:

1. \( (2x)(3x^2 - x) = 6x^3 - 2x^2 \)
2. \( (x^2 + 1)(6x - 1) = (x^2)(6x - 1) + 1(6x - 1) = 6x^3 - x^2 + 6x - 1 \)

Combine these results:

\[
f'(x) = (6x^3 - 2x^2) + (6x^3 - x^2 + 6x - 1)
\]

Combine like terms:

\[
f'(x) = 12x^3 - 3x^2 + 6x - 1
\]

Now, we need to evaluate \( f'(1) \):

\[
f'(1) = 12(1)^3 - 3(1)^2 + 6(1) - 1
\]

Calculate each term:

\[
f'(1) = 12 - 3 + 6 - 1 = 14
\]

Thus, the value of \( f'(1) \) is:

\[
\text{Answer: } 14

   \[

   \[
   \log_2 20 = \frac{\log 20}{\log 2}

   Now, \( \log 20 = \log (2 \times 10) = \log 2 + \log 10 \). So:

   \[
   \log_2 20 = \frac{\log 2 + \log 10}{\log 2}
   \]

   Simplifying this expression:

   \[
   \log_2 20 = \frac{\log 2}{\log 2} + \frac{\log 10}{\log 2} = 1 + \frac{\log 10}{\log 2}
   \]

   \[

   \[

   \]

4. **Calculate \( a \times b \):**

   We know \( b = \log 2 \). Therefore:

   \[
   a \times b = \left(1 + \frac{1 - \log 2}{\log 2}\right) \times \log 2
   \]

   Distribute \(\log 2\):

   \[
   a \times b = \log 2 + (1 - \log 2)
   \]

   Simplifying:

   \[

   \]

To solve this problem, we need to determine the acceleration of point P at the time when its direction of motion changes after it starts moving. The position of point P at time \( t \) is given by the function:

\[ x(t) = t^3 - \frac{3}{2} t^2 - 6t. \]

**Step 1: Determine the velocity function.**

The velocity \( v(t) \) is the derivative of the position function \( x(t) \) with respect to time \( t \). Therefore, we differentiate \( x(t) \):

\[ v(t) = \frac{d}{dt}(t^3 - \frac{3}{2} t^2 - 6t) = 3t^2 - 3t - 6. \]

**Step 2: Find the critical points where the velocity is zero.**

The direction of motion changes when the velocity changes sign, which occurs when the velocity is zero. We set the velocity function equal to zero and solve for \( t \):

\[ 3t^2 - 3t - 6 = 0. \]

Divide the entire equation by 3 to simplify:

\[ t^2 - t - 2 = 0. \]

This is a quadratic equation, which can be factored as:

\[ (t - 2)(t + 1) = 0. \]

Setting each factor equal to zero gives the solutions:

\[ t - 2 = 0 \quad \Rightarrow \quad t = 2, \]
\[ t + 1 = 0 \quad \Rightarrow \quad t = -1. \]

Since \( t \geq 0 \), we discard \( t = -1 \) and consider only \( t = 2 \).

**Step 3: Determine the acceleration function.**

The acceleration \( a(t) \) is the derivative of the velocity function \( v(t) \). Therefore, we differentiate \( v(t) \):

\[ a(t) = \frac{d}{dt}(3t^2 - 3t - 6) = 6t - 3. \]

**Step 4: Calculate the acceleration at \( t = 2 \).**

Substitute \( t = 2 \) into the acceleration function:

\[ a(2) = 6(2) - 3 = 12 - 3 = 9. \]

Thus, the acceleration of point P at the time when its direction of motion changes is 9.
